Chapter 1. Installation

Most features of CDI Schedule are installed automatically by including the cdi-schedule.jar and quatz libs in the web application library folder.

If you are using Maven as your build tool, you can add the following dependency to your pom.xml file:

        
            org.jboss.seam.faces
            seam-faces
            ${seam-faces-version}

Chapter 2. Scheduling jobs

CDI Schedule provides a complete API to schedule and manage jobs. To do that, you just need annotate the method you can schedule as following:

            @Schedule(jobName = "testJob", value = "0/1 * * * * ?", autoStart = true)
            public void onSchedule() {
                
            }

The @Schedule annotation has four atributes:

  • jobName
  • jobGroup
  • autoStart
  • value
